# Build Cordova

| Plataforma    | Comando                     |
| ------------- |:----------------------------:
| Android       | ```npm run build:android``` |
| iOS           | ```npm run build:ios```     |
| Windows       | ```npm run build:windows``` |

Passando options do [Ionic-App-Scripts](https://github.com/driftyco/ionic-app-scripts/blob/master/README.md) para alguns dos comandos listados basta adicionar ```--``` ap√≥s o comando desejado
ex: ```npm run build:android -- --prod```
